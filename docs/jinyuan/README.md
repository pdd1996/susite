# 金源电器样板站

金源电器是展站 V1 的首个样板客户，用于验证 B2B 制造企业展示站模板、内容结构和素材交付流程。

## 当前范围

- 当前 MVP 为 5 个路由页面：首页、产品中心、资质认证、关于我们、联系我们。
- 产品中心在同一页面中展示 5 个产品分类；独立分类路由不属于当前 MVP。
- 不包含留言表单、客户账号、CMS、后端运行时或正式域名上线流程。
- 可运行原型位于相邻的 `jinyuan-mvp` 工作区；其 localStorage、Base64 上传和模拟发布仅用于演示，不代表展站 V1 平台能力。
