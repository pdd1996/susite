# 金源电器官网 — 设计规范 & 视觉稿说明

| 项目 | 内容 |
|------|------|
| **版本** | Design v1.1 |
| **日期** | 2026-05-24 |
| **适用范围** | MVP 官网（5 Tab + PDF 下载） |
| **品牌参考** | 《金源2026版.pdf》产品选型手册 |

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **与 PDF 统一** | 网站是样本册的线上延伸，不能两套视觉 |
| **稳重专业** | B2B 制造业，给老总看，拒绝花哨 |
| **信息克制** | 短句、大字号、少段落，参数留给 PDF |
| **移动优先** | 销售微信发链接，手机浏览是主场景 |
| **无阴影** | 扁平设计，用边框和色块区分层级 |

## 2. 色彩系统

### 2.1 主色

| 名称 | 色值 | 用途 |
|------|------|------|
| **品牌红 Primary** | `#C41E3A` | Logo 圆、CTA 按钮、Tab 当前态、标题装饰线 |
| **品牌红 Hover** | `#A8182F` | 按钮/链接悬停 |
| **品牌红浅底** | `#FDF2F4` | CTA 区块背景、卡片 hover 底色 |

> 色值取自 PDF 中 BH Logo 红色，开发时可微调 ±5%。

### 2.2 中性色

| 名称 | 色值 | 用途 |
|------|------|------|
| **标题黑** | `#1A1A1A` | H1/H2、公司名称 |
| **正文灰** | `#4A4A4A` | 段落、描述 |
| **辅助灰** | `#8C8C8C` | 次要信息、页脚链接 |
| **占位灰** | `#BFBFBF` | 占位图、禁用态 |
| **分割线** | `#E8E8E8` | 区块分隔、卡片边框 |
| **区块背景** | `#F7F7F7` | 理念区、优势区、产品图底 |
| **页面背景** | `#FFFFFF` | 主背景 |
| **页脚背景** | `#1A1A1A` | Footer 深色底 |

### 2.3 色彩使用规则

```
✅ 红色用于：Logo、主按钮、当前导航、标题下短线（40×3px）
✅ 深色用于：页脚、首屏 Banner 遮罩
❌ 禁止：大面积红色背景、渐变按钮、彩虹色图标
❌ 禁止：box-shadow 阴影（全站扁平）
```

## 4. 间距系统

基于 **8px 网格**：

| Token | 值 | 用途 |
|-------|-----|------|
| `space-xs` | 4px | 图标与文字间距 |
| `space-sm` | 8px | 卡片内元素间距 |
| `space-md` | 16px | 卡片内边距、网格 gap |
| `space-lg` | 24px | 卡片之间 |
| `space-xl` | 48px | 区块内 padding（上下） |
| `space-2xl` | 80px | 区块之间留白 |

### 布局

| 属性 | 值 |
|------|-----|
| 内容最大宽度 | `1200px` |
| 内容区左右 padding | 桌面 `24px`，移动 `16px` |
| 导航栏高度 | `64px` |
| 移动端底部栏高度 | `52px` |

## 6. 组件规范

### 6.1 导航栏 Header

```
┌──────────────────────────────────────────────────────────────┐
│ [BH●] 金源电器    首页  产品中心  资质认证  关于我们  联系我们  [下载样本册] │
└──────────────────────────────────────────────────────────────┘
  64px 高 · 白底 · 固定顶部 · 滚动后加 1px #E8E8E8 底边
```

| 状态 | 样式 |
|------|------|
| 默认 Tab | `#4A4A4A`，14px |
| 当前 Tab | `#C41E3A`，14px，底部 2px 红线 |
| Hover | `#C41E3A` |
| 下载按钮 | 背景 `#C41E3A`，白字，padding `8px 16px`，圆角 4px |

### 6.2 按钮

| 类型 | 样式 | 用途 |
|------|------|------|
| **Primary** | 背景 `#C41E3A`，白字，hover `#A8182F` | 下载 PDF、一键拨号 |
| **Outline** | 边框 `#C41E3A`，字 `#C41E3A`，hover 浅红底 | 查看资质、了解更多 |
| **Ghost** | 无框，字 `#4A4A4A` | 次要操作 |
| **White Outline** | 首屏用：白字 + 半透明白框 | Banner 区「联系我们」 |

**尺寸：**
- 大：`padding: 12px 24px`（首屏 CTA）
- 中：`padding: 10px 20px`（区块 CTA）
- 小：`padding: 8px 16px`（导航栏下载）

**最小点击区域：** 44 × 44px

## 7. 页面视觉稿说明

### 7.1 首页

```
┌─ Header ─────────────────────────────────────────────┐
├─ Banner（280px 高）─────────────────────────────────┤
│  [深色左遮罩]                                        │
│  杭州金源电器有限公司                                 │
│  HANGZHOU JINYUAN ELECTRIC CO., LTD.               │
│  专注互感器研发制造                                   │
│  [下载样本册]  [联系我们]                              │
├─ 企业理念（#F7F7F7 底）──────────────────────────────┤
│  [质量] [科技] [管理] [服务]  四格                     │
├─ 产品中心（白底）────────────────────────────────────┤
│  5 张产品大类卡片，桌面端 3+2 排列                    │
├─ 核心优势（#F7F7F7 底）──────────────────────────────┤
│  4 条 icon + 短句                                    │
├─ 资质荣誉（白底）────────────────────────────────────┤
│  4 张证书缩略图 + [查看全部资质]                       │
├─ 联系摘要（#F7F7F7 底）──────────────────────────────┤
│  电话 + 地址 + 二维码                                 │
├─ Footer（#1A1A1A）──────────────────────────────────┤
```

**Banner 背景图建议：** 沿用 PDF 封面城市夜景风格，1920×600，左侧 40% 加深色遮罩。

### 7.2 产品中心

- 页面标题 + 装饰线 + 一句总述
- 5 个大类纵向排列，每类之间 `space-2xl`（80px）分隔
- 每类：H3 标题 → 1 段简介 → 系列卡片 3 列网格
- 底部 CTA 区块：浅红底 `#FDF2F4` + 下载按钮 + 电话

### 7.3 资质认证

- 全页白底
- 证书分类：企业资质 → 型式批准 → 体系认证 → 检验报告
- 每类 H3 + 大图/网格
- 大图最大宽 800px，居中

### 7.4 关于我们

- 企业简介：左文右图（如有厂房图）或纯文字
- 理念四格
- 应用行业 8 图标（来自 PDF 第 3 页）
- 定制能力文字块

### 7.5 联系我们

- 左：联系信息列表（电话可点击）
- 右：微信二维码 + 样本册二维码
- 下方：地图（高度 300px）
- 无表单

## 8. 移动端适配

| 元素 | 桌面 | 移动 |
|------|------|------|
| 导航 | 横排 Tab | 汉堡菜单 |
| Banner 高 | 280px | 200px |
| H1 | 32px | 22px |
| 产品网格 | 3 列 | 2 列或 1 列 |

**断点：**
- `< 768px`：移动端
- `768px – 1199px`：平板
- `≥ 1200px`：桌面

## 10. 交互规范

| 交互 | 行为 |
|------|------|
| Tab 切换 | 当前页红色 + 下划线，无页面刷新（SPA） |
| 产品卡片 hover | 边框 → `#C41E3A`，200ms ease |
| 证书点击 | Lightbox 放大，点击遮罩关闭 |
| 下载 PDF | 直接下载，新 Tab 打开 |
| 电话点击（移动） | `tel:0571-86817925` |
| 导航滚动 | 距顶 > 0 时加底边线 |
| 页面切换 | 滚动到顶部 |

**动画：** 仅 hover 过渡 200ms，不做 scroll 动画、不做 parallax。
